## ğŸš€ Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm

### Installation

1. **Clone the repository**
   ```bash
   git clone [repository-url]
   cd bark-battle
   ```

2. **Install client dependencies**
   ```bash
   cd client
   npm install --force
   ```

3. **Install server dependencies**
   ```bash
   cd server
   npm install --force
   ```

4. **Start the development servers**
   
   **Terminal 1 - Client:**
   ```bash
   cd client
   npm run dev
   ```
   
   **Terminal 2 - Server:**
   ```bash
   cd server
   npm run dev
   ```

5. **Open your browser**
   Navigate to `http://localhost:3000` to start barking!
